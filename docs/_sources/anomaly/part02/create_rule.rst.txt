알람 룰 만들기
----------------------------------------

대상 데이타 선정, 모델 트레이닝을 위한 범위 및 값 선정, 자동 트레이닝, 최적 모델의 선정, 알람 규칙 설정 등을 
순차적으로 수행하여 사용자가 원하는 알람룰을 쉽게 생성할수 있도록 지원한다.
